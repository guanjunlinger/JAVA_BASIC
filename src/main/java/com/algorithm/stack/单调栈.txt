单调栈:栈底->栈顶的元素具备单调性

性质:
单调递增栈:第一个比当前元素小的元素
单调递减栈:第一个比当前元素大的元素

经典例题:

1944. 队列中可以看到的人数
思路:
找到右边第一个比他高的人

768. 最多能完成排序的块II
思路:
后面的分块的最小值要大于等于前面的分块的最大值,计算每一个元素i之前的最大值和之后的最小值


321. 拼接最大数
思路:
dp[i][k]:在前i个数字中挑选k个可以获得的最大值
dp[i][k] = max({dp[i-1][k-1]+nums[i]},dp[i-1][k])

1793. 好子数组的最大分数
给你一个整数数组nums和一个整数k.一个子数组(i, j)的分数定义为min(nums[i], nums[i+1], ..., nums[j]) * (j - i + 1)
一个好子数组的两个端点下标需要满足i <= k <= j
请你返回好子数组的最大可能分数
思路:
1.寻找以nums[k]为最小值的好子数组(k1,k2)
2.以较大nums[k]值为最小值的好子数组,直至数据遍历完



907.子数组的最小值之和
给定一个整数数组arr,找到min(b)的总和,其中b的范围为arr的每个连续子数组
思路:寻找每个元素取最小值的区间
